##  Problem statement

While the Claude Plugin Marketplace is being developed, the broader engineering team needs structured guidance on how to effectively use AI development tools. Without curated resources, individual engineers must figure out best practices on their own, leading to inconsistent adoption and potential frustration with the tools.

This initiative bridges the gap between now and the marketplace project by providing immediate resources that help engineers get value from AI tools today.

##  Scope

**Must have:**| 

  * **AI Tool Tips Sharing Channel**
    * Dedicated Slack channel for sharing tips
    * Weekly curated tips
    * Coverage of both GitHub Copilot and Claude Code
  * **Use-Case Library / Prompt of the Week**
    * Shared wiki for proven prompts
    * Categories: 
      * code generation
      * debugging
      * testing
      * documentation
      * code review
    * Template for submitting new use cases
  * **Quick-Start Guide (Github Copilot)**
    * "Start Here" doc with 5 high-value use cases
    * Step-by-step setup instructions
    * Common pitfalls and troubleshooting

  * Video recordings of demos
