# Request upgrade to Claude Premium

Source: https://basis.atlassian.net/wiki/spaces/BET/pages/1607532591

If you are running into usage limits on the standard tier, you can request access to Claude Premium. <https://support.claude.com/en/articles/9266767-what-is-the-team-plan#h_b59203dff2>

####  Before requesting access, please ensure you review the following documentation to optimize your usage

  * <https://support.claude.com/en/articles/9797557-usage-limit-best-practices>

## Requesting Claude Premium

You (the employee who wants Claude Premium) should submit the request. After you submit the request, the tool will need your manager to approve the request.

  1. Navigate to the [Basis Atlassian Support portal](https://basis.atlassian.net/servicedesk/customer/portal/5)

  2. Select `Access`

  3. Select `Update Existing Access`

  4. Fill out the form using the following example

  5. A support ticket will be automatically generated

  6. Manager approves the request in the support ticket

  7. Wait for Basis Atlassin Support to complete the request ~a few hours

  8. Receive a confirmation email from Anthropic
