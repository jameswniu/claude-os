## Recruiting -> Interview -> Hire phases

Assuming the applicant passes each phase in the process, they will go through the following phases in our interview process:

  1. Initial phone screen by Talent Acquisition team (30 mins)
  2. Hiring manager phone screen (45 mins)
  3. At-home coding exercise (exercise should take a couple of hours to complete, ask the applicant to review the exercise then provide us with an estimated timeline)
  4. In-person interview at Centro
     1. Warm-up chat / Coding exercise Part 1 (1 hour)
     2. Coding exercise Part 2 (45 mins)
     3. Data modeling (1 hour)
     4. Wrap-up with Hiring Manager or Mike (45 mins)
  5. Interview Team Debrief (30 mins, same day as interviews if at all possible)
  6. Hiring Manager reference checks (~48 hours post debrief)
  7. Hiring Manager extends offer, get start date and verbal acceptance (~15 mins)
  8. T&D background check (~24 hours)
  9. Signed offer letter from applicant (~12-48 hours)
