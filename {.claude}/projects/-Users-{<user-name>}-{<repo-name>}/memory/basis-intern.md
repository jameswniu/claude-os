## Recruiting -> Interview -> Hire phases

Assuming the applicant passes each phase in the process, they will go through the following phases in our interview process:

  1. Initial phone screen by Talent Acquisition team (?? mins)
  2. Hiring manager phone screen (45 mins)
  3. Panel interviews:  

     1. Coding exercise Part 1 (45 mins)
     2. Coding exercise Part 2 (45 mins)
     3. Wrap-up with Hiring Manager (45 mins)
  4. Interview Team Debrief (30 mins, same day as interviews if at all possible)
  5. Hiring Manager extends offer, get start date and verbal acceptance (~15 mins)
  6. T&D background check (~24 hours)
  7. Signed offer letter from applicant (~12-48 hours)

  

Traits we're looking for:

  * Inquisitive
  * Self motivated
  * Responsible
  * Eager to learn
  * Detail Oriented
  * Ability to take criticism/suggestions
